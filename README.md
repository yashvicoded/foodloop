## FoodLoop: AI-Powered Food Waste Reduction ♻️

**FoodLoop** is an AI-powered system designed to reduce food waste in retail environments. This project was developed as part of a February 2026 hackathon.

### 📂 Project Structure

This repository is organized as a monorepo:

* **`foodloop-frontend/`**: The React/Vite web application built with Tailwind CSS.
* **`foodloop-backend/`**: The server-side logic and API, integrating with Firebase and Supabase.

---

### 🚀 Getting Started

#### 1. Clone the repository

```bash
git clone https://github.com/yashvicoded/foodloop.git
cd foodloop

```

#### 2. Setup Frontend

```bash
cd foodloop-frontend
npm install
npm run dev

```

#### 3. Setup Backend

```bash
cd ../foodloop-backend
npm install
npm run start

```

---

### 🛠️ Tech Stack

* **Frontend**: Vite, React, Tailwind CSS
* **Backend**: Node.js, TypeScript, Firebase, 
* Database: Firebase (Firestore + Auth)
* Core Logic: Rule-based expiry-based discount engine

---

### 💡 Features

* AI-driven inventory analytics
* Dynamic discount engine for expiring products
* Donation tracking for local food banks
